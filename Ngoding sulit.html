About Me

Selamat datang di dunia kreativitas digital, saya Set, seorang pengembang web dengan dedikasi tinggi dan hasrat untuk menciptakan pengalaman online yang luar biasa. Saya percaya bahwa setiap proyek memiliki potensi untuk menjadi sebuah karya seni yang dapat memberikan nilai tambah bagi pengguna.

Saya memiliki latar belakang yang kokoh dalam pengembangan web, dengan pengalaman lebih dari 10 tahun. Dari proyek-proyek sederhana hingga solusi-solusi kompleks, saya telah bekerja dengan beragam klien untuk mewujudkan visi dan tujuan mereka melalui platform online.

My Skills

Sebagai seorang pengembang web, saya menguasai keterampilan HTML, CSS, dan JavaScript untuk menciptakan antarmuka yang responsif dan dinamis. Kemampuan saya dalam menggunakan kerangka kerja seperti React dan Vue memperkuat pengembangan aplikasi yang inovatif. Keahlian desain dengan Figma dan manajemen proyek dengan GitHub menunjukkan keterampilan holistik saya dalam menyusun solusi web yang efektif dan estetis.

Html 5 Mark Up

Sebagai pengembang web yang berkomitmen, saya menguasai HTML sebagai keahlian inti. Dengan pemahaman mendalam tentang struktur markup dan tag HTML5, saya mampu menciptakan tata letak responsif, formulir interaktif, dan mengoptimalkan konten multimedia. Keahlian saya dalam HTML memastikan bahwa setiap proyek yang saya kerjakan memiliki dasar yang kuat.

CSS styling

Dengan keahlian mendalam dalam CSS, saya sebagai pengembang web dapat menciptakan antarmuka yang menarik dan responsif. Menguasai pemilihan warna, tata letak fleksibel, dan animasi CSS, saya mampu memberikan desain yang estetis dan fungsional. Keterampilan CSS saya memastikan bahwa setiap elemen pada situs web yang saya kembangkan memiliki penampilan yang konsisten dan profesional.

Javascript

Sebagai pengembang web dengan keterampilan JavaScript yang kuat, saya mampu menciptakan pengalaman interaktif yang dinamis. Memahami DOM dan AJAX, saya mengimplementasikan fungsionalitas yang responsif dan real-time. Keahlian dalam menggunakan kerangka kerja seperti React dan Vue memperkuat kemampuan saya dalam mengembangkan aplikasi web yang inovatif dan efisien.

Grapic design

Dengan keterampilan desain menggunakan Figma, saya sebagai pengembang web membawa elemen estetika ke setiap proyek. Mampu merancang tata letak intuitif dan antarmuka yang menarik, saya memastikan pengalaman pengguna yang optimal. Kombinasi keahlian Figma dan pengembangan web memungkinkan saya menghasilkan desain yang sesuai dengan kebutuhan fungsionalitas dan keindahan visual.

Grapic design

Dengan keterampilan desain menggunakan Canva, saya sebagai pengembang web menghadirkan kreativitas visual ke setiap proyek. Kombinasi keahlian Canva dan pengembangan web memungkinkan saya menyampaikan pesan dengan estetika yang kuat, menciptakan pengalaman online yang tak terlupakan.

Tool teamwork

Dengan keterampilan GitHub yang kuat, saya sebagai pengembang web memastikan kolaborasi yang efisien dan kontrol versi yang akurat dalam setiap proyek. Mengelola repository dengan baik, saya menjaga kode sumber terorganisir dan memfasilitasi kerja tim. Pemahaman saya tentang fitur seperti isu (issues) dan penarikan (pull requests) membuat kontribusi terbuka dan penyelesaian masalah lebih efektif.
